# Intelligence Platform Dashboard

A Streamlit-based dashboard for managing **Cybersecurity incidents**, **Data Science datasets**, and **IT tickets**. This project is designed for monitoring, CRUD operations, and visualization.

---

---

## Features

### Cybersecurity Dashboard

- Add, update, delete incidents
- Visualize incidents by type, severity, and status
- Interactive charts using Plotly

### Data Science Dashboard

- Add, update, delete dataset metadata
- Visualize dataset sizes and record counts by category
- Interactive charts using Plotly

### IT Ticket Dashboard

- Add, update, delete tickets
- Visualize tickets by priority and status
- Easy CRUD operations with Streamlit expanders

### Authentication

- Simple login/logout functionality
- Session-based login checks
